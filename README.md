## Project Overview
MFI-Pro is a specialized frontend proof-of-concept developed to streamline internal workflows for Microfinance Institutions (MFIs). Unlike consumer-facing landing pages, this application focuses on the Core Banking UI: handling complex financial calculations, data-heavy tables, and reactive state management for loan officers.

Built with Angular and TypeScript, this project demonstrates a "Business-First" approach to software development, prioritizing data integrity and reactive performance.
Utilities: RxJS for reactive data streams

## Tech Stack
- Framework: Angular 21 (Standalone Components)
- Language: TypeScript (Strict Mode)
- UI Library: Angular Material / Tailwind CSS
- Charts: Ng2-Charts / Chart.js
- State Management: RxJS / Signals
